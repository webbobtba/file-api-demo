# file-api-demo
file API demo with parcel &amp; tailwind

## Install

1. Clone repo:
```js
git clone https://github.com/webbobtba/file-api-demo.git
```

2. Go into directory:
```js
cd file-api-demo
```

3. Get dependencies:
```js
npm install
```

4. Start parcel:
```js
npm start
```

5. build for production (includes custom tailwind build):
```js
npm run build
```
